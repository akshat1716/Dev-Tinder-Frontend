## 📖 About The Project

In a world full of talented developers, finding the right person to collaborate with can be a challenge. **devTinder** aims to solve this problem by providing a fun, intuitive, and effective platform for developers to connect.

Users can create a profile showcasing their skills, projects, and interests. The app then presents profiles of other developers, allowing users to **swipe right** to express interest or **swipe left** to pass. When two users mutually swipe right, it's a match! They can then start a conversation and explore potential collaborations.

> ⚠️ **Backend Repository**:

## ✨ Features

- 👤 **User Authentication**: Secure sign-up and login functionality.  
- ✏️ **Profile Customization**: Create a detailed developer profile with skills, experience, project links (GitHub, portfolio), and social media.  
- ↔️ **Swiping Mechanism**: Classic Tinder-style swiping to browse and match with other developers.  
- 🤝 **Matching System**: Get notified when you have a mutual match.  
- 💬 **Real-time Chat**: *(Planned Feature)* A built-in chat to communicate with your matches.  
- 📱 **Responsive Design**: A clean and modern UI that works seamlessly on desktop and mobile devices.

## 🚀 Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

Make sure you have **Node.js** and **npm** installed on your Device.

```bash
npm install npm@latest -g

Installation
Clone the repository:


git clone https://github.com/carti0459/devTinder-web-Frontend.git
Navigate to the project directory:


cd devTinder-web-Frontend
Install dependencies:


npm install
Start the development server:


npm run dev
The app will run at http://localhost:5173 (or the next available port).
